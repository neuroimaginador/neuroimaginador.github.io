Up & Running with blogdown

Sep 14, 2017 @dominlopez  View source 
Diapositivas Pre-work GitHub R-Ladies PDX GitHub

Resumen

Join me for a workshop to get you up and running with your own personal website. If you are looking to start your own site, come learn how to do it for free using using R, RStudio, and the blogdown R package.

Fecha
Sep 14, 2017
00:00
Evento
Portland R User Group Workshop
Localización
Oregon Health & Science University, Portland, Oregon

We will cover:

  • The blogdown R package
  • Hugo themes
  • Adding content to personalize your site
  • Posting
  • Don’t want to blog? / just want to knit?
  • Customizing themes
  • Troubleshooting common problems
  • Deployment options and recommendations
  • Domain options available through RStudio/rbind

If you are planning on attending this workshop, please click on the HTML link above for how to prepare before the workshop.
